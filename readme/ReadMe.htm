1. Server Requirements

  • Operating System: Linux, Windows

  • Web Server: Apache

  • PHP 5 or higher

  • MySQL 5 or Higher

  • Mod_rewrite

  • Cron Jobs

  • CURL

  • GD library

  • Facebook Application Id

  • Facebook Application Secret Key

  • Twitter Application Id

  • Twitter Application Secret Key

  • Cloudinary Cloud name

  • Cloudinary API Key

  • Cloudinary API Secret

  • Google Map Key

  • Ioncube Loader

2. Installation

Note: You must get the valid cloudinary API keys before starting installation

1. Upload all files in the archieve to the root directory or subdirectory (i.e yoursite.com or yoursite.com/install).


2. Change the permission of the following folders and files to 0777:

/install
/install/config.php [After the installation revert the file permission]
/app/config
/app/config/config.php [After the installation revert the file permission]
/files
/files/cache
/files/timthumb.php

3. Run the /install directory to install the system (i.e yoursite.com/install)


4. Open the /php.ini file and place following line
zend_extension="/home/dropinn/public_html/ioncube/ioncube_loader_lin_5.2.so"
But remove following lines with your Absolute root path- /home/dropinn/public_html/


5. Run the /ioncube/loader-wizard.php file to install the Ioncube loader (i.e yoursite.com/ioncube/loader-wizard.php)

Manual Installation

1.Edit the file app/config/config.php and edit your mysql details in the following lines:

$config['hostname'] = "hostname";
$config['db_username'] = "mysqluser";
$config['db_password'] = "mysqlpass";
$config['db'] = "mysqlDB";

2.$config['base_url'] = "Site URL "; (i.e yoursite.com/site)

3. In PhpMyAdmin, import the file install/install.sql to create the database structure.


4. Change the permission of the following folders and files to 0777:

/images
/images/users

5. Open the /php.ini file and place following line
zend_extension="/home/dropinn/public_html/ioncube/ioncube_loader_lin_5.2.so"
But remove following lines with your Absolute root path- /home/dropinn/public_html/


6. Run the /ioncube/loader-wizard.php file to install the Ioncube loader (i.e yoursite.com/ioncube/loader-wizard.php)

 

7. To access the admin panel /administrator (i.e yoursite.com/administrator), login with username: what you entered password: what you entered on the login page



8. Set the Cron job for every 1 hour for get the latest currency values from online.
Cron URL : SITE_URL/currency_converter



9. Set the Cron job for every 1 hour for to check the reservation request is expired or not.
Cron URL : SITE_URL/cron/expire



10. Set the Cron job for every 1 hour for calendar auto sync.
Cron URL : SITE_URL/cron/calendar_sync

11. Set the Cron job for every 1 hour for reservation status.
Cron URL : SITE_URL/cron/accept_status

12. Set the Cron job for every 1 hour for after checkin status changed to checkout.
Cron URL : SITE_URL/cron/after_checkin
